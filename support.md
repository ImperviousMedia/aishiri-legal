## Common Issues

### Purchases / Lore Mode
- If subscription plans aren’t showing, confirm you’re signed into the App Store and have an active internet connection.
- If you already subscribed, use **Restore Purchases** from the Lore Mode paywall.

### Account / Sign-In
- If you forgot your password, use the password reset option on the sign-in screen.
